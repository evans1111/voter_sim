Description:

A program that allows a user to create a directory of politicians and voters.
Users will be able to Create, List, Update and Delete both voters and politicians from the directory.

Voters and Politicians are both People with names.
Voters however, have a political affiliation of either Liberal, Conservative, Tea Party, Socialist, or Neutral, while politicians have a party affiliation either Democrat or Republican.


How To Run:
To start the game, run the file run.rb
